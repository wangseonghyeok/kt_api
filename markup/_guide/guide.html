기본 가이드

1. HTML

Document Type Definition : HTML5 / Charset : utf-8로 표기한다.

각 언어별 html태그에 Language Attribute를 반드시 표기한다.

브라우저는 Chrome을 기준으로 하되 MS edge / firefox 를 필수 브라우저로, IE10 이상을 옵션 브라우저로 수용할 수 있는 웹표준 코딩을 기본으로 한다.

웹접근성을 준수하여 작업한다. (OpenWAX, W3C markup Validation 사용)

화면모드는 1920x1080을 기준으로 한다.

요소와 속성은 소문자로 표기되어야 하며, 모든 태그는 종료 태그를 갖는다.

속성 "값"은 항상 따옴표로 감싸주어야 한다.
